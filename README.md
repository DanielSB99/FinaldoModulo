# Pokémon Database (Laravel Project)

## About This Project
This is a backend-focused academic project developed as the final module assignment for my Information Systems Programming course. The goal was to build a database and web interface to manage and display Pokémon games and their associated development studios.

## Features
* **Manual Data Management:** Creation and management of Pokémon games and studios using custom backend logic, without relying on external APIs.
* **Relational Database:** Designed and implemented the database structure to link games to their respective studios.
* **Backend Architecture:** Built entirely using PHP and the Laravel framework.

## Technologies Used
* PHP
* Laravel
* MySQL
